  • $http://obsidian.md - automatic!$
    • 指向網路連結，以網址顯示，$僅為排版美觀
  • $[obsidian](http://obsidian.md)$
    • 指向網路連接，以中括號內內容顯示

Obsidian URL

Obsidian URI 通常採用以下形式：

$$obsidian://action?param1=value¶m2=value$$

action表示執行操作，value為參數 比如打開筆記：

$$[標註的筆記名，想給的代號](obsidian://open?path=筆記絕對路徑)$$

注意中英文括號、冒號 舉例： 打開高數 其中，path接檔案絕對路徑

  • 打開某庫，不打開具體檔案只打開資料夾不顯示效果 $$[name](obsidian://open?vault=庫資料夾名稱/庫ID)$$ 打開筆記資料夾
  • 打開Math資料夾下的高數，.md副檔名可不寫 高數

Others

  • URL中有空格時需要轉義為$\%20$

    • 上面%20的百分號%需要轉義，%

  • 或者用<>將有空格內容包圍

  • 引用

    $$> 引用內容$$

    $$\ 引用來源$$

    eg

時間是一場有去無回的旅行，好的壞的都是風景。 \ ——《歲月神偷》金玟岐

  • 行內程式碼 include<stdc++.h>

  • 程式碼塊 Obsidian 會將六個反引號括起來的內容識別為程式碼塊，並將其按程式碼格式顯示。程式碼塊支援語法高亮功能，你可以在程式碼塊的第一行（即開頭處的反引號後）宣告語言來使用特定的語法高亮。

#include<stdc++.h>
using namespace std;
int main(){
    int a=1;
    for(int b=2;b<10;b++){
        a+=b;
        cont<<a<<endl;
    }
    return 0;
}
