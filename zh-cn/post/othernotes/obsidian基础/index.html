外部链接

指向库中笔记/图片

  • $http://obsidian.md - automatic!$
    • 指向网络链接，以网址显示，$仅为排版美观
  • $[obsidian](http://obsidian.md)$
    • 指向网络连接，以中括号内内容显示

Obsidian URL

Obsidian URI 通常采用以下形式：

$$obsidian://action?param1=value¶m2=value$$

action表示执行操作，value为参数 比如打开笔记：

$$[标注的笔记名，想给的代号](obsidian://open?path=笔记绝对路径)$$

注意中英文括号、冒号 举例： 打开高数 其中，path接文件绝对路径

  • 打开某库，不打开具体文件只打开文件夹不显示效果 $$[name](obsidian://open?vault=库文件夹名称/库ID)$$ 打开笔记文件夹
  • 打开Math文件夹下的高数，.md拓展名可不写 高数

Others

  • URL中有空格时需要转义为$\%20$

    • 上面%20的百分号%需要转义，%

  • 或者用<>将有空格内容包围

  • 引用

    $$> 引用内容$$

    $$\ 引用来源$$

    eg

时间是一场有去无回的旅行，好的坏的都是风景。 \ ——《岁月神偷》金玟岐

  • 行内代码 include<stdc++.h>

  • 代码块 Obsidian 会将六个反引号括起来的内容识别为代码块，并将其按代码格式显示。代码块支持语法高亮功能，你可以在代码块的第一行（即开头处的反引号后）声明语言来使用特定的语法高亮。

#include<stdc++.h>
using namespace std;
int main(){
    int a=1;
    for(int b=2;b<10;b++){
        a+=b;
        cont<<a<<endl;
    }
    return 0;
}
