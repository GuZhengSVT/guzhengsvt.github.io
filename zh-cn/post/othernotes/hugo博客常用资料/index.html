Hugo 常用命令

命令作用说明
hugo version查看版本显示当前 Hugo 安装版本
hugo new site <项目名>创建新站点会生成 Hugo 的目录结构
hugo new post/<文件名.md>创建新文章会在 content/post/ 下生成文章并附带默认 Front Matter
hugo server启动本地服务启动本地预览，默认地址 http://localhost:1313
hugo server -D启动并显示草稿-D 参数显示 draft: true 的文章
hugo构建网站将 Markdown 内容生成静态文件到 public/ 目录
hugo -D <输出目录>构建包含草稿用于发布前的全站构建（含草稿）
hugo --minify压缩构建文件构建时压缩 HTML/CSS/JS，减小体积
hugo config查看配置信息显示当前站点的配置信息
hugo list drafts列出草稿查看所有设置了 draft: true 的文章
hugo list future列出未来发布的文章date 设置在当前时间之后的文章
hugo list expired列出过期文章expiryDate 已过期的文章
hugo --gc垃圾回收清理过时资源，如缩略图缓存
